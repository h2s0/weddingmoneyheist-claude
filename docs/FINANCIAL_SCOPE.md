# 금융 기능 범위

## 제품 목적

이 서비스는 사용자가 본인 명의 증권계좌의 정보를 한 화면에서 확인하고, 사용자가 직접 선택한 매수법의 설정과 가상 계획을 관리하는 도구다. 특정 금융상품의 매수·매도 권유, 투자자문, 일임 운용을 제공하지 않는다.

## MVP 포함 범위

- 카카오 OAuth를 통한 사용자 식별과 서버 세션
- 사용자가 직접 발급받은 증권사 API 자격증명의 서버 측 검증과 암호화 저장
- 계좌, 예수금, 보유종목, 평가금액, 손익, 거래내역 조회
- 조회 결과의 대시보드 표시와 동기화 상태 안내
- 매수법 카탈로그 열람, 사용자 파라미터 입력, 설정 저장
- 실제 주문을 발생시키지 않는 회차·예산·가상 주문 계획 계산

## MVP 제외 범위

- 주식·ETF·파생상품의 실제 주문, 정정, 취소
- 예약 주문, 조건 주문, 자동매매, 서버 스케줄 주문
- 사용자 자금을 보관하거나 이동하는 기능
- 수익 보장, 종목 추천, 매수·매도 시점 추천
- 사용자 설정 없이 포트폴리오를 변경하는 기능
- 증권사 자격증명 또는 접근 토큰을 브라우저에 저장하는 방식

## 사용자 통제 원칙

- 사용자는 증권사 연결 상태, 마지막 동기화 시각, 데이터 오류를 확인할 수 있어야 한다.
- 사용자는 자격증명을 교체하고 연결을 해제하며 저장된 전략을 비활성화할 수 있어야 한다.
- 전략 결과는 가상 계획임을 화면에서 명확히 표시하고 실제 체결 가능성이나 수익을 보장하지 않는다.
- 향후 주문 기능은 현재 MVP의 연장으로 자동 활성화하지 않고 별도 법률·보안·운영 검토와 프로젝트 승인을 거친다.

## 데이터와 감사

- 증권사 비밀정보는 backend에서만 취급하고 KMS 기반 envelope encryption을 적용한다.
- 사용자별 데이터 접근 경계를 모든 query와 DB 제약에서 검증한다.
- 로그인, 연결 검증, 동기화, 키 교체, 연결 해제를 비밀값 없는 감사 이벤트로 기록한다.
- 계정 삭제 시 세션, 증권사 연결, 암호문, 전략 설정과 관련 개인정보를 함께 삭제하는 절차를 제공한다.
